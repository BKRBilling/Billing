Accounting and Inventory Software Project

Description:  A comprehensive application that integrates financial tracking and inventory management, streamlining business operations for efficient management.

For More Details Technologies Used:
- VB.NET: For building the application and backend logic
- SQL Server: For database management and data storage
- Windows Forms: For creating the user interface

Features:
- User-friendly dashboard for financial overview and inventory status
- Modules for managing sales, purchases, and expenses
- Inventory tracking with stock level alerts
- Reporting tools for financial analysis and inventory audits

QR Code Generator Project

Description:  A simple QR code generator that takes a user input string and generates a QR code image.


For More Details Technologies Used:
- HTML: For structuring and creating the QR code generator interface
- CSS: For styling and layout
- JavaScript: For generating the QR code image

Features:
- User input field to enter the string to be encoded
- Generate button to create the QR code image
- QR code image display area
- Download button to save the QR code image
